Sobre Simu

Simu es un proyecto creado para simplificar el cálculo de quincenas. Nace como una solución práctica para trabajadores que quieren claridad y organizacion sobre sus ingresos sin depender de cálculos complejos.

¿Cómo funciona?

1. Ingresá datos

Cargás tu información laboral básica.

2. Procesamiento

El sistema calcula automáticamente.

3. Resultado

Obtenés una estimación clara.

Preguntas Frecuentes

¿Los resultados son exactos?

No, son estimaciones basadas en cálculos aproximados.

¿Necesito registrarme?

Por el momento No, podés usar la herramienta libremente.

¿Funciona para cualquier trabajo?

Está optimizado para ciertos casos, pero es adaptable.
