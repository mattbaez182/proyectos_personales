Simulá tu sueldo en segundos

Simu es una herramienta online para simular y calcular tu sueldo de manera rápida, facil y segura. Ideal para conocer descuentos, totales y con resultados en tiempo real. Calculá tu sueldo de forma rápida, clara y profesional.

🚀 ¿Cómo usar Simu?
Simu es un simulador de cálculo de quincena diseñado para darte una estimación clara, rápida y detallada de tu ingreso.

🧾 🧠 ¿Qué datos te vamos a pedir?
Para realizar el cálculo, vas a tener que completar los siguientes campos: 🔹 Datos básicos
Quincena → Elegí si querés calcular la *primera* o *segunda quincena*
Categoría → Tu categoría laboral (esp, 1, 2, 3, 4)
Horas trabajadas → Cantidad de horas trabajadas en la quincena
Adicionales → Cantidad de adicionales que te correspondan
Feriados → Cantidad de días feriados trabajados
Antigüedad → Años de antigüedad en la empresa
Ausencias → Cantidad de faltas (importante para premios)

🔹 Datos especiales (solo segunda quincena)
Horas de la primera quincena → Necesarias para calcular el premio de producción.
🔹 Datos de descuento
Afiliado al sindicato → * ✔ Sí → se aplica 21% * ❌ No → se aplica 19%

⚡ 🧮 ¿Qué calcula Simu?
Simu procesa automáticamente:
💰 Haberes:
* Sueldo base
* A cuenta futuro aumento (ACFA)
* Adicionales
* Feriados
* Presentismo (si no hay ausencias)
* Premio de producción (solo en segunda quincena)
* Antigüedad

Descuentos
* Porcentaje según afiliación (19% o 21%)
* En ambos descuentos estan integrados jubilacion, ley 19032 y obra social.

📊 📌 Resultado Vas a ver un resumen claro con:
* ✔ Detalle completo de haberes
* ✔ Total bruto (sin descuento)
* ✔ Descuento aplicado
* ✔ Neto estimado a cobrar
* Si hay ausencias, algunos premios no se aplican

🔥 Consejo
Para obtener un resultado más preciso:
✔ Ingresá todos los datos correctamente
✔ Revisá si corresponde cargar datos de la primera quincena
✔ Indicá correctamente tu condición (afiliado o no)

🔒 Politica y Privacidad

En Simu la privacidad y seguridad de tus datos son una prioridad.
✔ No almacenamos información personal
✔ No guardamos los datos ingresados
✔ No utilizamos bases de datos actualmente
✔ No compartimos información con terceros

Todos los cálculos se realizan "en tiempo real", utilizando únicamente los datos que ingresás en el momento.

🧠 ¿Qué significa esto?
* La información que cargás (horas, categoría, antigüedad, etc.)
👉 no se guarda ni se registra en ningún sistema.

* Cada simulación es completamente "temporal"
👉 los datos se pierden al actualizar o cerrar la página
